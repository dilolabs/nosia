# Medietilsynet AI Platform - Feature List

## Core AI Functionalities (4.3)

- **Chat with LLM** - with ability to define roles/instructions for the conversation
- **RAG (Retrieval-Augmented Generation)** - use internal documents to improve answers
- **Export to Word and PowerPoint** - convert AI responses to documents
- **Reasoning mode** - model breaks down complex instructions and shows its thinking process
- **Image generation** - create images via AI
- **AI-assisted web search** - search the web with AI help
- **AI-assisted internal data search** - search internal documents/systems
- **Excel/CSV data analysis** - analyze and work with spreadsheet data
- **AI Agents** - autonomous programs that plan and execute multi-step tasks without user input at each step

## Document & Data Integration (4.4, 5.1, 5.2)

- **Connect to SharePoint Online** - search and use SharePoint data
- **Connect to OneDrive for Business** - access OneDrive files
- **Connect to Exchange (email)** - search email data
- **Connect to Salesforce** - integrate with Salesforce data
- **Connect to Acos WebSak+** - integrate with case management system
- **Permission-aware search** - only show data user has access to
- **Multi-source search** - search across multiple systems simultaneously
- **Search filtering** - filter by document type, date, department
- **File preview** - preview documents from search results
- **Add files to AI for analysis** - attach files from search to AI conversation

## Prompt & Instruction Management (3.1, 3.2)

- **Save prompts/instructions** - store successful prompts for reuse
- **Share prompts with colleagues** - make prompts available to team
- **Organize prompts** - tag, categorize, search saved prompts

## Chatbots (7.1, 7.2, 7.3)

- **Create chatbots** - configure chatbots with specific instructions
- **Chatbot knowledge base** - attach documents to chatbots
- **Internal chatbots** - deploy bots for employee use
- **External/public chatbots** - deploy bots for public access
- **Chatbot access controls** - control who can use which bots (specific employees, all employees, public)
- **Easy documentation updates** - simple way to update chatbot knowledge

## Admin Controls & Configurability (4.2, 6.1)

- **Multiple AI models** - support different LLMs simultaneously
- **Model selection** - users can choose which AI model to use
- **Admin model control** - admins choose which models are available to users
- **Feature flags** - turn individual AI features on/off
- **Role-based access** - assign features to specific user groups
- **Gradual rollout** - enable features for pilot users before full launch
- **Access groups** - manage user groups for feature distribution

## Analytics & Statistics (10.1)

- **Active user tracking** - number of active users over time
- **Feature usage statistics** - which features are used most (reasoning, web search, saved prompts, etc.)
- **Prompt reuse tracking** - how often saved prompts are used
- **Usage reports** - reports for admins to evaluate adoption and training needs

## User Experience (1.1)

- **Intuitive interface** - easy to use for varying technical skill levels
- **Low barrier to entry** - quick to learn and use
- **Consistent navigation** - easy to find features after periods of non-use
- **User-friendly design** - accessible and clear

## Security & Privacy (8.1, 9.1, 9.2)

- **Self-hosted deployment** - runs on Medietilsynet's own servers
- **GDPR compliance** - meets data protection requirements
- **EU/EØS data hosting** - all data stays in EU/EØS region
- **No vendor data sharing** - data not shared with AI providers
- **No model training on customer data** - their data not used to train models
- **User permission respect** - system respects existing access controls
- **Encryption support** - respects data encryption
- **No information leakage** - prevents data from leaving the organization
- **ISO27001 or equivalent** - security management system

## Documentation & Support (2.1, 2.2)

- **Technical documentation** - for IT staff and operations team
- **User guides** - for employees using the system
- **Updated documentation** - kept current with system changes
